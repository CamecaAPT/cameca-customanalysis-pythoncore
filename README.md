## Description
Core class library that provides tools for using Python to create extensions for AP Suite.
This is not an extension by itself, it it designed to be used as a toolkit for developing other extensions that use Python.
Leverages Python.NET for interoperability.
Designed to be extensible to support various Python distributes. Intially only configured to use Anaconda.
